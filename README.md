# 🎓 Student Attendance Management System

This project is a web-based **Student Attendance Management System** built using Flask and MySQL.  
It is designed to automate the traditional attendance process in educational institutions and provide a simple, efficient, and digital way to manage student attendance records.

---

> ##  Project Overview

Attendance management is an important part of any academic institution.  
Manual attendance systems are time-consuming, error-prone, and difficult to maintain for long-term records.

This system solves these issues by providing a **centralized web application** where teachers can:

- Mark student attendance digitally
- View attendance history
- Generate attendance reports
- Manage student records efficiently

The system ensures better accuracy, transparency, and easy access to data.



## 🚀 Features

- Secure login system for teachers/admin
- Add, update, and manage student details
- Mark daily attendance
- View attendance records in tabular format
- Generate attendance reports
- Forgot password functionality
- Clean and user-friendly dashboard

---

## 🧠 System Architecture

The system follows a simple **client-server architecture**:

- **Frontend:** HTML, CSS, JavaScript (User Interface)
- **Backend:** Flask (Python handles logic & routing)
- **Database:** MySQL (Stores student and attendance data)

Flow of system:
User → Frontend → Flask Backend → MySQL Database → Response → UI Display
